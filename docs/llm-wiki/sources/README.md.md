## Overview
MAA (MaaAssistantArknights) is an assistant for the game Arknights, based on image recognition technology. It automates daily tasks like combat, infrastructure management, recruitment, and more.

## Key Features
- **Combat**: Sanity management, drop recognition, and data upload.
- **Infrastructure**: Smart operator switching and optimal efficiency calculation.
- **Recruitment**: Automated recruitment and data upload.
- **Resource Management**: Recognition of materials and export to planning tools.
- **Daily Tasks**: Collecting rewards, visiting friends, etc.
- **Roguelike Automation**: Automated grinding in roguelike modes.
- **JSON Tasks**: Custom task execution via JSON files.
- **Multi-language Support**: C, Python, Java, Rust, Golang, etc.

## Technology Stack
- **Language**: C++20
- **Platform**: Windows, Linux, macOS
- **Recognition**: Image recognition based.
